Avant d’utiliser le Template_ODT, vous devez télécharger la police Playfair Display. Pour cela : 

1) Copier le .zip "playfair-display.zip" sur votre machine. 

2) Extraire le dossier "playfair-display" du .zip

3) Sélectionner toutes les polices (fichier en .ttf) 

4) Cliquer droit sur la sélection puis installer.



Vous pouvez maintenant utiliser le Template_ODT !
